Notus

A Drone Companion App

✈️ Notus is an Android app for drone pilots to send messages, generate reports and access vital information

🤝 I developed the app on behalf of a local charity, within which the Drone Team are still using Notus

Development Process

🔎 I interviewed charity members and conducted extensive research to devise a list of user requirements

👨‍💻 I decomposed the app's features, created various drafts and then started programming the solution

🧪 I tested the app using test tables and a range of inputs during the iterative development process

📝 Finally, I sent the final app to the stakeholders and attached a user survey to collect feedback

🌐 The repository for Notus is currently private, but feel free to check out my GitHub profile:

Languages and Tools

☕ I developed the Android app in the Java programming language, utilising a range of OOP techniques

🔗 I leveraged several APIs to provide accurate location and weather data directly in the app

📖 I documented the entire process in a Word document containing over 300 pages

🧠 This project enabled me to learn a lot about software development
